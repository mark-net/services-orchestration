# Распределённая система из четырех сервисов.

Архитектура  
![схема взаимодействия сервисов](scrn/distributed.png)  
![результат в консоли](scrn/cmd.png)

Компоненты:  
**API Gateway (Fastify, порт 3000)** - одна точка входа, пересылает запросы в основной сервер.  
**Main Server (Go, порт 8080)** - управляет всем, собирает данные от микросервисов и отдает клиенту.  
**Service1 (Go, порт 8081)** - делает строку заглавными буквами.  
**Service2 (Go, порт 8082)** - переворачивает строку.

Почему так выбрал:  
Fastify для Gateway - потому что он очень быстрый и при этом простой.  
Go для остальных сервисов - потому что работает быстро, мало памяти ест и код пишется легко и надёжно.  
Main Server не просто пересылает, а собирает ответы - чтобы Gateway занимался только маршрутизацией, а микросервисы оставались простыми и независимыми.  
HTTP/REST вместо gRPC - для прототипа так проще писать и проверять, не нужны дополнительные инструменты.  
Синхронные вызовы - для обработки текста это нормально, асинхронность тут только усложнит.

Как работает:  
1. Клиент отправляет POST с `{"text": "hello"}` в Gateway  
2. Gateway пересылает в Main Server  
3. Main Server одновременно дергает оба микросервиса:  
   - Service1 отдаёт `"HELLO"`  
   - Service2 отдаёт `"olleh"`  
4. Main Server собирает все вместе  
5. Клиент получает:  
```json
{
  "original": "hello",
  "uppercase": "HELLO",
  "reverse": "olleh"
}
```

Масштабирование  
Если нагрузка вырастет — можно запускать много копий любого сервиса, поставить балансировщик, добавить кэш в Main Server или потом перейти на gRPC внутри.

Технологии  
- Go 1.25.5 — main server и микросервисы  
- Fastify 4.28 — API Gateway  
- HTTP/REST — синхронное общение  
- Git — версии

Запуск  
```bash
# Терминал 1 - Service1
cd service1 && go run main.go
# Терминал 2 - Service2
cd service2 && go run main.go
# Терминал 3 - Main Server
cd main-server && go run main.go
# Терминал 4 - Gateway
cd gateway && npm install && node server.js
```

Тест  
```bash
curl -X POST http://localhost:3000 \
  -H "Content-Type: application/json" \
  -d "{\"text\":\"hello\"}"
```
